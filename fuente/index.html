@use

Para usar los elementos definidos en un primer módulo en un segundo hay que usar @use en el segundo salvo en las funciones, en cuyo caso SASS es permisivo y permite usarlas en algunos casos cuando se importan a través de un archivo principal (_main.scss). Así, si se importan las funciones y el fichero que las usa en el principal, el que las usa las puede utilizar aunque no tenga @use funciones

@use "sass:map"; @use "sass:math"; @use "sass:string"; @use 'variables'; @use 'media-query';

Variables

Las variables pueden ser globales o locales:

Definición de variables globales

$color-primario: #3498db; $tamaño-fuente: 16px; body { font-size: $tamaño-fuente; background-color: $color-primario; }

Definición de variables locales

.navbar { $color-fondo: #2c3e50; background-color: $color-fondo; }

Interpolación

Permite insertar dinámicamente valores en cadenas de texto o selectores:

Ejemplo 1: En comentarios

$autor: "Chiquito de la calzada"; /* Web desarrollada por #{$autor} */

Ejemplo 2: En nombres de clases

$color: 'red'; .button-#{$color} { background-color: $color; }

Ejemplo 3: En propiedades CSS

$prefijo: 'font'; $tamaño: 'size'; #{$prefijo}-#{$tamaño} { font-size: 16px; }

Ejemplo 4: Combinaciones de valores

$base: 'color'; $valor: 'blue'; #{$base}-background: $valor;

Ejemplo 5: En URLs

$image-path: 'images/logo.png'; background-image: url('#{$image-path}');

Funciones

Ver el archivo _funciones.scss.

@function escala-tipografica($base, $factor) { @return $base * $factor; } h1 { font-size: escala-tipografica(16px, 1.5); // Resultado: 24px } h2 { font-size: escala-tipografica(16px, 1.25); // Resultado: 20px }
